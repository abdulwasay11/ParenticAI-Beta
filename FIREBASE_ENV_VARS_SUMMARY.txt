HOW TO SET IN VERCEL:
================================================================================

1. Go to your Vercel project dashboard
2. Navigate to Settings → Environment Variables
3. Add each variable above:
   - Key: The variable name (e.g., REACT_APP_FIREBASE_API_KEY)
   - Value: The value shown above (or get new values from Firebase Console)
   - Environment: Select all (Production, Preview, Development)
4. Click Save
5. Redeploy your application

================================================================================
HOW TO GET VALUES FROM FIREBASE:
================================================================================

1. Go to https://console.firebase.google.com/
2. Select your project
3. Click gear icon (⚙️) → Project settings
4. Scroll to "Your apps" section
5. Click on your web app (or create one)
6. Copy values from the config object

See FIREBASE_ENV_VARS.md for detailed instructions.
